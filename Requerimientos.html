Minecraft: Requisitos del Sistema

Todo lo que necesitas saber para jugar Minecraft

Requisitos para Java Edition (PC)

Mínimos

  • Sistema operativo: Windows 7 o posterior, macOS 10.12 Sierra o posterior, Linux
  • Procesador: Intel Core i3-3210 / AMD A8-7600 APU o equivalente
  • Memoria RAM: 4 GB
  • Tarjeta gráfica: Intel HD Graphics 4000 o AMD Radeon R5 series con OpenGL 4.4
  • Almacenamiento: 4 GB de espacio libre (1 GB para el juego + 3 GB para archivos)
  • Java: Java 8

Recomendados

  • Sistema operativo: Windows 10, macOS 10.15 Catalina o posterior, Linux actualizado
  • Procesador: Intel Core i5-4690 / AMD A10-7800 o equivalente
  • Memoria RAM: 8 GB o más
  • Tarjeta gráfica: GeForce 700 Series o AMD Radeon Rx 200 Series o superior
  • Almacenamiento: SSD con 8 GB de espacio libre
  • Java: última versión de Java

Requisitos para Bedrock Edition (Windows)

Mínimos

  1. Sistema operativo: Windows 10 version 14393.0 o superior
  2. Arquitectura: x64 con soporte para DirectX 11
  3. Procesador: Intel Celeron J4105 / AMD FX-4100 o mejor
  4. Memoria RAM: 4 GB
  5. GPU: Intel HD Graphics 4000 o AMD Radeon R5 series
  6. Almacenamiento: 1 GB de espacio libre

Recomendados

  1. Sistema operativo: Windows 10 20H1 o posterior
  2. Arquitectura: x64 con soporte para DirectX 12
  3. Procesador: Intel Core i5-7400 / AMD Ryzen 3 1300X o superior
  4. Memoria RAM: 8 GB
  5. GPU: NVIDIA GeForce GTX 1060 / AMD Radeon RX 580 o superior
  6. Almacenamiento: SSD con 4 GB de espacio libre

Requisitos para Consolas y Móviles

Minecraft está disponible en múltiples plataformas:

  • Xbox One y Series X/S
  • PlayStation 4 y 5
  • Nintendo Switch
  • Dispositivos Android con al menos 2 GB de RAM
  • Dispositivos iOS con iOS 10 o superior

Notas Adicionales

Los requisitos pueden variar según la cantidad de mods, shaders y la complejidad de los mundos.

Para servidores multijugador, se recomiendan conexiones de internet estables con al menos 10 Mbps de velocidad.
